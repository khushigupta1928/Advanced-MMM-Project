# Advanced-MMM-Project
Built an Advanced Marketing Mix Modeling (MMM) solution using Python, Linear Regression, Adstock transformation, and marketing feature engineering. Evaluated model performance, interpreted sales drivers, generated business insights, and designed an interactive Power BI dashboard for decision-making.

Advanced Marketing Mix Modeling (MMM)
Overview

This project focuses on identifying the factors that influence sales using Marketing Mix Modeling (MMM). Linear Regression was used to measure the impact of marketing, pricing, promotions, distribution, and market variables on sales performance.

Tools Used
Python
Pandas
NumPy
Scikit-Learn
Matplotlib
Power BI

Key Steps
Data Cleaning
Exploratory Data Analysis (EDA)
Adstock Transformation
Feature Engineering
Linear Regression Modeling
Model Evaluation
Business Insights
Power BI Dashboard

Key Findings
Distribution was the strongest sales driver.
Promotions positively impacted sales.
Media channels contributed positively to sales.
Higher prices were associated with lower sales.

Dashboard
The project includes an interactive Power BI dashboard to visualize sales trends, marketing performance, and key MMM insights.
